# Getting started

PyOps runs locally alongside Factorio. It can plan against any synced mod set, with
additional views for Pyanodons features when that data is available.

The full first-run guide is in progress. For now:

1. Download the installer for your platform from the
   [latest GitHub release](https://github.com/ApocDev/pyops/releases/latest).
2. Start PyOps and open **Settings → Game data**.
3. Sync the Factorio installation and mod set you want to plan against.
4. Open **Blocks** and create your first production block.

PyOps stores its projects locally. Factorio is required for the initial game-data sync,
but the companion mod and live in-game connection are optional.
